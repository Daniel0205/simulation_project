El pseudocodigo del proyecto se enecuentra en el archivo "Pseudo.py" en la raiz de este repositorio

Para ejecutar la simulacion de cada punto,se deben tener instalados los modulos simpy y numpy del lenguaje de progrmacion Python
y para correrlo se debe ubicar en la respectiva carpeta del punto y se debe correr con el comando python3 <Nombre_del_archivo>
por ejemplo:

```bash
$ cd PuntoA // para ingresar a la carpeta
$ python3 PuntoA.py //para correr el proyecto
```

Para instalar los modulos, se puede hacer uso del instalador de modulos de python, llamado pip, corriendo los siguientes comandos:

```bash
$ pip3 install numpy
$ pip3 install simpy
```
